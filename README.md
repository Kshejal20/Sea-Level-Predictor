# 🌊 Sea Level Predictor

Predict future global sea levels using historical data and linear regression. This project analyzes long-term sea level measurements and visualizes historical trends alongside future predictions.

> **Project:** Sea Level Predictor
> **Certification:** freeCodeCamp – Data Analysis with Python

---

# 🚀 Project Overview

This project analyzes historical global sea level data from **1880 to 2013** and builds linear regression models to forecast future sea levels through the year **2050**.

Two regression lines are generated:

* A regression line using the entire dataset.
* A regression line using data from the year 2000 onward.

These models help compare long-term trends with more recent changes in sea level.

---

# 📂 Dataset

**File:** `epa-sea-level.csv`

The dataset contains annual measurements of global average sea levels.

| Column                   | Description                       |
| ------------------------ | --------------------------------- |
| Year                     | Observation year                  |
| CSIRO Adjusted Sea Level | Global average sea level (inches) |

---

# 📊 Visualization

The project creates a scatter plot of historical sea level data and overlays two linear regression lines.

### Historical Data

* Displays annual sea level observations.

### Long-Term Regression

* Uses all available data (1880–2013).
* Predicts sea level through 2050.

### Recent Trend Regression

* Uses data from 2000 onward.
* Highlights how recent sea level trends compare with the historical average.

# 📊 Project Workflow

1. Load historical sea level data.
2. Create a scatter plot of annual measurements.
3. Fit a linear regression model using the complete dataset.
4. Predict sea levels through 2050.
5. Fit a second regression model using data from 2000 onward.
6. Compare long-term and recent sea level trends.
